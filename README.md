# FlowerCost Pro

Professional florist profit analysis and staff training software. Stop losing money on beautiful arrangements with real-time cost tracking and staff training tools.

## Features

- **Real-time Cost Tracking**: Track wholesale costs and retail pricing in real-time
- **Staff Training Mode**: Train seasonal staff to maintain profit margins
- **Universal POS Integration**: Works with any POS system via copy/paste
- **Arrangement Recipes**: Save and reuse profitable arrangement formulas
- **Profit Analytics**: Detailed business insights and performance tracking
- **Role-Based Access**: Owner, Manager, and Staff access levels

## Getting Started

### For Florists
1. Visit [FlowerCost Pro](https://flowercostpro.com)
2. Start with the demo data to explore features
3. Add your own products and markup settings
4. Train staff using Staff Training Mode

### For Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Technology Stack

- **Frontend**: React 18 + TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Build Tool**: Vite
- **Deployment**: Netlify

## Business Model

FlowerCost Pro helps florists:
- Maintain consistent profit margins
- Train seasonal staff quickly
- Prevent pricing mistakes during busy periods
- Analyze business performance
- Work with any existing POS system
